# Mario Platformer (Godot Edition)

Классический 2D-платформер, вдохновлённый оригинальной Super Mario Bros, полностью написанный на **Godot Engine**. В игре 4 уникальных уровня, культовые враги, препятствия и плавная физика движения. Проект собран под системы Android, Windows и Aurora OS.

## Установка и запуск

### Android
1. Скачай `.apk` из раздела [Releases](https://github.com/top-it-090304/Flycat4er_Team/releases)
2. Разреши установку из неизвестных источников.
3. Установи и запусти приложение на Android устройстве.

### Aurora OS
1. Скачай `.rpm` из раздела [Releases](https://github.com/top-it-090304/Flycat4er_Team/releases)
2. Разреши установку из неизвестных источников.
3. Установи и запусти приложение на Aurora устройстве.

## Сборка из исходников
1. Установи Godot `4.4.1` или новее
2. Открой проект  
3. Установи необходиый пакет файлов для сборки проекта
4. `Проект → Экспорт` → выбери пункт Android или Aurora OS (обязательно на Linux)
5. Нажми `Export Project`

## Управление
| Платформа      | Движение      | Прыжок / Действие |
|----------------|---------------|-------------------|
| Android / Aurora OS | Сенсорное управление стрелочками | Стрелочка вверх |
| Windows     | Управление с помощью кнопок WASD | Пробел |

## Особенности
- **4 уровня** с уникальным дизайном и прогрессирующей сложностью
- **3 типа врагов**: Грибочки, черепахи, босс Боузер
- **Классические препятствия**: Пропасть, растения, огненные снаряды, платформы, различные постройки
- **Кроссплатформенность**: полная поддержка сенсорного управления на различных типах устройств
- **Оптимизация**: стабильный геймплей на мобильных устройствах Android

## Технологии
- **Движок:** Godot Engine `4.4.1` или новее
- **Язык:** GDScript
- **Экспорт:** Android (APK), Aurora OS (RPM)
